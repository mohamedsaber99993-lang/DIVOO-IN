## 🛠️ ⚡ خطوات التشغيل والنشر السريع
### 📱 1. التشغيل المحلي (Termux / PC)
قم بفتح الطرفية ونفذ الأوامر التالية بالتتابع:

# استنساخ المستودع
git clone [https://github.com/dev-ABOODI-cpu/DIVOO-IN.git](https://github.com/dev-ABOODI-cpu/DIVOO-IN.git)
cd DIVOO-IN
# تثبيت الحزم والمكتبات الاعتمادية
npm install
# بدء تشغيل البوت
npm start

### ☁️ 2. النشر على منصة Railway
 1. قم بربط حساب Railway الخاص بك بمستودع جيت هاب هذا (DIVOO-IN).
 2. سيقوم السيرفر بعمل بناء تلقائي للمشروع (Build).
 3. توجه إلى سجلات المنصة (**Logs**)، وستجد كود التحقق والربط المكون من 8 خانات قد ظهر أمامك تلقائياً؛ قم بنسخه وربطه من هاتفك مباشرة!
